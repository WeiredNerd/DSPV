在DSPV系统里价值仅仅在作品里。
符号本身不代表价值，只负责标记，传播，帮助作品找到对的人，帮助别人知道你做过什么。

这听起来好像很不错，但你会觉得，这是不是一种**Idealism**?

假设你是一个读过Simulacra, 也知道福柯“权力无处不在”的家伙，你可能会陷入悲观。

那么我来提供一个新的角度：

权力和Simulacra的强大不是问题，甚至还能被当作工具使用。
问题在于，我们不会用。只会被这些工具牵着走。

为什么？
我们明明进化出了前额叶，能够进行复杂的长线规划和系统设计，为什么现在还困在：

资源分配逻辑被Simulacra劫持，导致个体无法做自己，只能将自己硬生生改造成资源分配体系想要的样子来换取一点资源的困境？

这不是什么结构性困境。
这是”用马桶塞子装水喝而不是用来疏通马桶“的问题。

**我们依旧从现状入手。**

平台创作者必须要研究用户画像，数据，市场套路，到处收集点子来追求流量增长；
社交媒体平台靠算法黑箱赚钱。管理层吃肉，执行曾吃肉渣。

这是个什么生态呢？
这是一方吸另一方的血，而另一方还甘愿这么做的问题。
因为另一方不这么做，就连肉渣都没有了。

我们来剖析一下这一套系统的价值流动问题。

对于创作者来说，价值就是流量。因为在这套系统里，流量自动和变现画上等号。
但是流量归根结底是用户的注意力。用户的注意力有限，必须要想办法争夺这种资源。
这是典型的零和系统：我抢到资源（流量），你就没有了。我赢了，你输了。

而联系到平台资本和执行层（包括维护平台运营的工作团队和创作者）的关系也是这套零和游戏。
由流量转变的流水就那么多，你们执行层多了，我资本方就少了。
所以我需要用各种宣传手段（也就是制造出各种Simulacra）来掩盖这一层逻辑。

这种零和问题可以放在教育领域的资源分配逻辑上。

但其实实际中，零和游戏的每一个生态位根本不需要Simulacra理论，因为Simulacra是自己长出来的。就跟地心引力一样，一开始就存在，只是你给它命名了。

总结起来这个零和生态是什么呢？
所有人（包括收割方）被困在零和游戏的Simulacra里，没有人意识到自己被Simulacra耍。
以往我们会攻击收割方。
但我要说这不是收割方的问题（系统角度，不是伦理角度。剥削是真实的不需要解释），这是整个零和游戏Simulacra的系统运作方式问题。

**系统运作本身不是问题。
但是系统在吃人。这就是问题。
The system is not the problem--The problem is that system is eating people.**

但是，
### 这依旧不是最根本的问题###

最根本的问题其实不在社会学和哲学。
是在**人类进化史**上。
